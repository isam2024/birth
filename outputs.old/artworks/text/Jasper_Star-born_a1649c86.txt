# Untitled

by Jasper Star-born

**"Echoes in the Rotting Log"**

```
   _____
  /      \
 /        \
|  decay  |
 _______/
  |       |
  |  roots  |
  |_______|
   ____
  |       |
  |  sap    |
  |  flows  |
  |_______|
```

This piece represents the ephemeral nature of life, where growth and decay are intertwined. The rotting log is a symbol of impermanence, while the roots and sap flowing through it represent resilience and transformation.
